Leaflet + Geoapify Places API sample: categorized search rendered as map markers

 
Zoom: 14

How this demo works

  • Pick the categories you want via the buttons; each toggle updates the categories parameter sent to the Geoapify Places API.
  • Pan or zoom the map (zoom ≥ 13) to trigger a Places request for the current bounds and plot the results as markers.
  • The visible bounds are subdivided into small fragments, so results stream in chunk by chunk for a smoother, more responsive UX.
